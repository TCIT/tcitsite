HCMFront Web development

HCMFront

HCMFront

Is a strong application which holds features for recruiting, selecting and managing human resources.

What is fascinating about this web application is its data model. It gathers know how from the best human resource consultors and experts in the field. With this, this application works fine for every industry.

Its most powerful tool might be the performance evaluation, it is just great. It allows for 360 degree evaluation, peer-to-peer, external-evaluators, bosses and subordinates for each employee of a company. It also permits to evaluate automatically the competences and skills for the position profile of each employee, this way all of the relevant competences and skills for an employee can be evaluated given their profile. It also also allows for customizing specific competences, skills and questions for an evaluation. Su herramienta más potente podría ser la evaluación de rendimiento, que es sencillamente genial. Permite una evaluación de 360 grados, cara a cara, evaluadores externos, jefaturas y subordinados para cada empleado de la compañía. También permite evaluar automáticamente las competencias y habilidades requeridos para cada perfil de cargo con respecto a cada empleado, de modo que todas las competencias y habilidades relevantes de un empleado puedan ser evaluadas, dado un determinado perfil. También permite personalizar preguntas, habilidades y competencias específicas para una evaluación
